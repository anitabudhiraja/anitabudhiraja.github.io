# Anita Budhiraja - Personal Portfolio Website

Welcome to the repository for my personal portfolio website. I am **Anita Budhiraja**, Scientist-E, Associate Dean – Academics, and Head of Department (Training) at NIELIT Chandigarh/Ropar, with over 25+ years of experience in AI, Machine Learning, Blockchain, IoT, and Emerging Technologies.

🔗 **Live Website:** [https://anitabudhiraja.github.io](https://anitabudhiraja.github.io)

## 🚀 Features

This portfolio is designed as a single-page application (SPA) with a strong focus on clean design, performance, and user experience.

- **Fully Responsive Design:** Optimized for desktops, tablets, and mobile devices with custom breakpoints.
- **Dark/Light Mode Toggle:** Built-in theme switcher for optimal viewing preferences.
- **Interactive Navigation:** Sticky navbar with active state tracking and a mobile-friendly hamburger menu.
- **Scroll Reveal Animations:** Content gracefully fades and slides into view using the Intersection Observer API.
- **Dynamic Project Filtering:** Easily filter portfolio projects by categories (Government, Research, Training).
- **Animated Statistics:** Number counters that animate from zero to their target when scrolled into the viewport.
- **No Dependencies:** Built entirely without heavy frontend frameworks (like React or Bootstrap) for maximum speed and simplicity.

## 🛠️ Technologies Used

- **HTML5:** Clean, semantic structure.
- **CSS3:** Custom CSS variables, Flexbox, CSS Grid, keyframe animations, and modern media queries.
- **JavaScript (Vanilla):** DOM manipulation, Intersection Observers, custom filtering logic, and event handling.
- **Typography:** 'Cormorant Garamond' and 'DM Sans' (via Google Fonts).

## 📂 Website Sections

1. **Hero:** Introduction, quick summary, and key statistics.
2. **About Me:** Professional background, demographic details, and resume download link.
3. **Expertise:** Core competencies across AI/ML, Data Science, Blockchain, AR/VR, and Academic Leadership.
4. **Career Journey:** Interactive timeline of professional milestones at NIELIT.
5. **Research & Publications:** Scholarly contributions, Springer publications, and international journals.
6. **Featured Projects:** Grid of software systems and government initiatives (with filtering capability).
7. **Highlights & Achievements:** Key recognitions, awards, and milestones.
8. **Testimonials:** Reviews from colleagues, directors, and students.
9. **Contact:** Reach out via the simulated contact form or direct links.

## 💻 Local Setup

To run this project locally, no build tools or package managers are required. Simply clone the repository and open the file!

```bash
# Clone the repository
git clone https://github.com/anitabudhiraja/anitabudhiraja.github.io.git

# Navigate to the project directory
cd anitabudhiraja.github.io

# Open index.html in your preferred web browser
# (Alternatively, use the "Live Server" extension in VS Code for hot reloading)
```
